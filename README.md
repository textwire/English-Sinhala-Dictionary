# English-Sinhala_Dictionary
*More than 45,000 English words with their Sinhala translations*

## Information
- english-sinhala_dictionary.ods is an OpenDocument Spreadsheet (.ods) created with LibreOffice Calc.
- This Spreadsheet contain a list of English words and their Sinhala translations.
- **Don't forget to Fork this Repo and contribute to maintain the wordlist.**

## Fonts (For Sinhala translations)
- [Iskoola Pota](http://www.sinhalafonts.org/fonts/13091/iskoola_potha_unicode.html)
- [Malithi Web](http://www.sinhalafonts.org/fonts/13092/malithi_web.html)

*Both of these fonts are fully compatible.*
